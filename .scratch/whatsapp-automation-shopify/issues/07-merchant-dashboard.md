## What to build

Build the merchant-facing dashboard UI. This is where merchants see the value of the app. It displays message statistics, recovered revenue, and allows them to toggle features on/off.

## Acceptance criteria

- [x] A dashboard page is accessible from the Shopify App navigation
- [x] It displays: total messages sent this month, remaining free quota, delivery rate
- [x] It displays: total recovered orders and total recovered revenue (from attribution data)
- [x] It displays: click-through rate for messages
- [x] Toggle switches are present to enable/disable Order Confirmation and Abandoned Cart features
- [x] A message history list shows recent messages with status (sent/delivered/failed)
